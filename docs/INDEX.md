# docs Index

## Purpose

Canonical repository knowledge base for agents and humans. This directory holds
the long-lived, versioned sources of truth for architecture, product behavior,
engineering standards, plans, references, and generated inventories.

## File conventions

- Markdown files only.
- Top-level files define cross-cutting rules, contracts, and navigation.
- Topic-specific knowledge belongs in subdirectories with their own `INDEX.md`.
- Generated artifacts live only under `generated/` and are refreshed by script.
- If knowledge is important for future agent runs, encode it here instead of
  leaving it in chat, tickets, or memory.
